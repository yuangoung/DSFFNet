To avoid insufficient complementarity between multimodal observations, we propose a dual-stream feature fusion network (DSFFNet) for high-resolution RTS susceptibility mapping using GaoFen-7 (GF-7), without relying on handcrafted feature and manually variables.
DSFFNet efficiently exploits the complementary information and extracts spectral-spatial contextual features and multiscale terrain representations through visual state-space (VSS) modeling and wavelet-convolutional terrain encoding.
